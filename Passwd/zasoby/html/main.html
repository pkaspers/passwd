0.1

Niniejsza dokumentacja zawiera opis klasy OpcjeWywolania. W obiekcie tej klasy przechowywany jest wynik parsowania opcji programu opisanego w zadaniu nr 9.

Z punktu widzenia korzystania z tej klasy istotne są tylko dwie metody, a mianowicie OpcjeWywolania::JakSortowac() oraz OpcjeWywolania::JakSzukac(). Najprostszy przykład ich wykorzystania w funkcji, która musi się znaleźć w module, aby dostarczony zalążek skonsolidował się poprawnie. 

 void PrzetworzListe_Uzytkownikow_i_Grupy(const OpcjeWywolania& Opcje)
 {
  cout << endl;
  cout << "Wywolanie przetwarzania listy uzytkownikow." << endl << endl;
  cout << "  Rodzaj sortowania: " << Opcje.JakSortowac() << endl;
  cout << "    Rodzaj szukania: " << Opcje.JakSzukac() << endl;
  cout << endl;
  cout << "              Login: \"" << Opcje.WezLogin() << "\"" << endl;                 
  cout << "                 Id: " << Opcje.WezIdUzytkownika() << endl;                   
  cout << endl;  
 }

Tę postać funkcji można użyć do przetesowania wywołania programu z różnymi opcjami, np. 

   ./uzytkownicy -s login -z 10421
